# Ventajas y desventajas de LaTeX

## Objetivo de la sesión

Al finalizar este tema, el estudiante será capaz de:

* Identificar las principales ventajas y desventajas del uso de LaTeX
* Comparar LaTeX con procesadores de texto tradicionales
* Decidir en qué situaciones es conveniente utilizar LaTeX

---

## 1. Ventajas de LaTeX

### 1.1 Calidad tipográfica

LaTeX produce documentos con una **calidad tipográfica profesional**, especialmente en textos largos, ecuaciones matemáticas y documentos académicos formales.

Las ecuaciones, márgenes, espaciados y títulos mantienen una apariencia uniforme en todo el documento.

---

### 1.2 Separación entre contenido y formato

En LaTeX el autor se enfoca en **el contenido**, mientras que el sistema se encarga del formato.

Esto permite:

* Cambiar el estilo completo del documento sin reescribir el texto
* Mantener consistencia visual
* Reducir errores de formato

---

### 1.3 Manejo automático de referencias

LaTeX permite:

* Numerar figuras y tablas automáticamente
* Crear referencias cruzadas
* Gestionar bibliografías con archivos `.bib`

Esto es especialmente útil en tesis y artículos científicos.

---

### 1.4 Ideal para documentos largos

LaTeX es muy estable en documentos extensos como:

* Tesis
* Libros
* Reportes técnicos

No presenta los problemas comunes de desorden de formato que aparecen en procesadores de texto tradicionales.

---

### 1.5 Herramienta gratuita y multiplataforma

LaTeX es:

* Gratuito
* Compatible con Windows, macOS y Linux
* Utilizado internacionalmente

Esto lo convierte en una herramienta estándar en el ámbito académico.

---

## 2. Desventajas de LaTeX

### 2.1 Curva de aprendizaje inicial

Al inicio, LaTeX puede resultar intimidante debido a:

* Uso de comandos
* Ausencia de edición visual directa

Sin embargo, esta dificultad disminuye rápidamente con la práctica.

---

### 2.2 No es WYSIWYG

En LaTeX no se ve el resultado final mientras se escribe.

Es necesario:

* Compilar el documento
* Revisar el PDF generado

Esto puede resultar incómodo para usuarios principiantes.

---

### 2.3 No siempre es necesario

Para documentos simples y cortos (por ejemplo, una carta o un informe de una página), el uso de LaTeX puede ser innecesario.

---

## 3. ¿Cuándo conviene usar LaTeX?

Se recomienda usar LaTeX cuando:

* El documento es largo
* Incluye ecuaciones, tablas o figuras complejas
* Requiere referencias bibliográficas
* Debe cumplir normas académicas estrictas

No se recomienda cuando:

* El documento es muy corto
* Se necesita una edición rápida sin formato complejo
